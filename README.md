# Fokir — Personal Portfolio Website

## Overview
**Fokir** is a personal portfolio website built using **HTML, CSS, and JavaScript**. It showcases personal information, services, portfolio projects, testimonials, a blog section, and a contact form in a modern and responsive layout.

🔗 Live Demo: https://marwaas3d.github.io/Fokir/

---

## Features
- Responsive and mobile-friendly design
- Interactive elements using JavaScript
- Organized sections for personal info, services, portfolio, and contact
- Clean and modern UI

---

## Sections
1. **Home** – Introduction and personal info
2. **About** – About yourself
3. **Services** – Services you provide
4. **Portfolio** – Showcase of your projects
5. **Testimonial** – Client feedback
6. **Blog** – Articles or posts
7. **Contact** – Contact form

---

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design
- Google Fonts and Font Awesome (optional)
